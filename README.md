# MyAIAgent

An AI research assistant built with Python, LangChain, and Google Gemini. This project explores AI agents, tool calling, and LLM-powered research workflows.

## Features

- Google Gemini integration
- LangChain agent framework
- Wikipedia search
- DuckDuckGo web search
- Research output saving
- Environment variable support with `.env`

## Tech Stack

- Python
- LangChain
- Google Gemini
- DuckDuckGo Search
- Wikipedia API
- Pydantic

## Acknowledgements

This project started from the YouTube tutorial *"Build an AI Agent From Scratch in Python - Tutorial for Beginners"* (by Tech with Tim), adapted to use Google Gemini.
